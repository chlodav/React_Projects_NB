# Phone Catalog

A responsive React and TypeScript phone catalog application with search, category filters, product previews, and a contact form. The project is built with Vite and optimized for fast loading and smooth user interaction.

## Live Preview

Live preview is not deployed yet. You can deploy the project to GitHub Pages, Netlify, or Vercel after the repository is ready.

## Design Reference

This project uses a custom responsive layout optimized for clarity, performance, and mobile-friendly browsing.

## Technologies Used

- React 18
- TypeScript
- Vite
- Sass (SCSS)
- Modern WebP image support

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/chlodav/project-name.git
cd project-name
```

2. Install dependencies:

```bash
npm install
# or
# yarn install
```

3. Run the project locally:

```bash
npm start
# or
# yarn start
```

4. Build for production:

```bash
npm run build
```

## Features

- Live search and category filtering for product browsing
- Responsive layout with mobile-first styling
- Smooth anchor navigation and hover interactions
- Lazy-loaded images for faster performance
- Contact form with current-page submission behavior

## Available Scripts

- `npm start` — run the app in development mode
- `npm run build` — create a production build
- `npm run lint` — run formatting and lint checks
- `npm run style-format` — format SCSS files
- `npm run format` — format TypeScript files
